Этап 1: 
Описание:

таблицы:
question: id, text(текст вопроса), type(чекбокс, радиобатон)
answer: id, questId(к какому вопросу относится), text(текст ответа), isCorrect(да\нет)
categories: id, parentId(родительский пункт), text(Название пункта), href(ссылка на пункт)
questToCategory: questId, categoryId

код:
реализуется на PHP 7.0.0., с нуля.

Этап 2:

АПИ:
- categories.getList(); // список имеющихся категорий, всё дерево пунктов по сути.
- categories.check();	// отметить пункт
- categories.uncheck(); // снять отметку с пунтка

- quiz.start();			// начать тест
- quiz.getOne();		// запросить один вопрос
- quiz.submitAnswer();	// отправить ответ
- quiz.getResult();		// запросить результат.
